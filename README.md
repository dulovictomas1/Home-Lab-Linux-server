# Home Lab / Linux Ubuntu server

A personal self-hosted Ubuntu Server project focused on learning Linux system administration, web server management, secure application deployment and reverse proxy configuration.

This project documents the setup and configuration of a home server running on Ubuntu Server inside VirtualBox. The server is used to host Laravel applications, manage Nginx virtual hosts, automate deployments and securely expose services to the internet using Cloudflare Tunnel.

## Features

- Ubuntu Server installation
- SSH remote administration
- Nginx web server configuration
- PHP & Composer environment
- MariaDB database server
- Laravel application deployment
- Virtual Hosts configuration
- Systemd service management
- Cloudflare Tunnel integration
- DNS configuration
- SSL/TLS via Cloudflare
- Git based deployment workflow
- Basic Linux server administration

## Technologies

- Ubuntu Server
- Nginx
- PHP
- Composer
- MariaDB
- Laravel
- Bash
- Systemd
- Cloudflare Tunnel
- Git

## Project Goals

The goal of this project is to gain practical experience with Linux server administration and deployment workflows by building a fully functional home server from scratch.

The server is used as a development environment for hosting Laravel applications and experimenting with networking, automation, reverse proxies and secure remote access.

## Highlights

- Configured an Ubuntu Server from scratch
- Deployed Laravel applications manually
- Configured Nginx virtual hosts
- Connected a local server to the internet with Cloudflare Tunnel
- Managed services with systemd
- Configured DNS and reverse proxy routing
